Papers
------

RankSEG is based on the following research papers:

JMLR 2024
~~~~~~~~~

The foundational paper introducing the RankSEG framework:

.. code-block:: bibtex

   @article{dai2024rankseg,
      title={RankSEG: A Consistent Ranking-based Framework for Segmentation},
      author={Dai, Ben and Wang, Zixun},
      journal={Journal of Machine Learning Research},
      volume={24},
      number={272},
      pages={1--41},
      year={2024}
   }

**Paper Link:** https://www.jmlr.org/papers/v24/22-0712.html

NeurIPS 2025
~~~~~~~~~~~~

Extended work on RankSEG methods:

.. code-block:: bibtex

   @inproceedings{dai2025rankseg,
      title={RankSEG: Adaptive Pixel Classification for Segmentation},
      author={Dai, Ben and Wang, Zixun},
      booktitle={Advances in Neural Information Processing Systems},
      year={2025}
   }

**Paper Link:** https://openreview.net/forum?id=4tRMm1JJhw
